# Validate Brand

This folder contains the working brand system for Validate.

Validate is an AI-powered product investment reasoning engine. The brand identity is intentionally minimal, monochrome, and evidence-driven.

## Core Concept

**Evidence becomes decision.**

The selected logo direction uses stacked evidence lines converging into a clear decision arrow.

## Contents

```text
/brand
  /logo
  /exports
  BRAND.md
  ASSET_EXPORTS.md
```

## Primary Files

- `BRAND.md` - Brand system, usage rules, tone, palette, and logo guidance.
- `ASSET_EXPORTS.md` - Export plan for SVG, PNG, favicon, app icon, and social assets.

## Logo Direction

The approved direction is the **evidence-to-decision mark**:

- stacked horizontal evidence lines
- converging into a right-facing decision point
- monochrome only
- modern SaaS / AI platform feel
- simple enough for app icon and favicon use

## Brand Attributes

- Clear
- Rigorous
- Modern
- Trustworthy
- Evidence-driven
- Minimal

## Usage Guidance

Use the horizontal logo lockup for most product and documentation surfaces.

Use the icon-only mark for:

- favicons
- app icons
- compact navigation
- small UI placements

Use the badge form for:

- social profile images
- dark-mode icon treatments
- app icon variants

## Current Constraint

The brand system is intentionally black and white.

Do not introduce color, gradients, sparkles, mascots, or generic AI symbols unless the brand direction is formally revised.

## Status

This brand system is a working draft and should evolve alongside the product.

Before making major visual changes, verify that the change still supports Validate's core philosophy:

> Validate turns evidence into better product decisions.
